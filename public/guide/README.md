# Coinbase API Guide Images

This directory contains annotated screenshots used by the Nexus AI onboarding chat to guide users through setting up their Coinbase API keys.

## Required Images

| File | Description |
|------|-------------|
| `coinbase-api-1-login.png` | Annotated screenshot of Coinbase Cloud login page → pointing to "API" in left sidebar |
| `coinbase-api-2-create.png` | Annotated screenshot of API page → "Create API Key" button circled |
| `coinbase-api-3-permissions.png` | Annotated screenshot of Create Key form → "View" and "Trade" permissions circled |
| `coinbase-api-4-copy.png` | Annotated screenshot showing where to copy the API Key Name and Private Key |

## How to Create

1. Log into https://cloud.coinbase.com
2. Navigate to the API section
3. Take screenshots at each step
4. Add red circles/arrows using any image editor (Preview, Snip & Sketch, etc.)
5. Save as PNG in this directory with the filenames above

## Dimensions
- Recommended: 1200×800px or similar desktop resolution
- Will be displayed inline in the chat at max-width: 100%
